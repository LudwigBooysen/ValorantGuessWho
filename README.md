# Valorant Guess Who

A browser-based Guess Who style game featuring agents from VALORANT.

## Features

* Randomly selects a VALORANT agent
* Ask questions and eliminate possibilities
* Simple and lightweight
* Runs entirely in the browser
* No installation required

## Tech Stack

* HTML
* CSS
* JavaScript

## Running Locally

1. Clone the repository
2. Open `index.html` in your browser

Alternatively, host the files using GitHub Pages, Vercel, or Netlify.

## Project Structure

```text
index.html
cards.json
scripts/
styles/
```
